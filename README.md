# Portfolio

UI/UX Designer Portfolio - Tharshan Mahalingam

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This portfolio is configured for GitHub Pages deployment. See `DEPLOY_NOW.md` for detailed deployment instructions.
